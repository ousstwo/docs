## Get set up

1. Clone this repo to your machine.
2. Install the Mintlify CLI: `npm i -g mint`
3. From the folder with `docs.json`, run: `mint dev`
4. Open `http://localhost:3000` to preview updates as you type.

<Tip>The preview reloads automatically whenever you save an MDX file.</Tip>

## Make your first edit

<Steps>
  <Step title="Update branding">
    Open `docs.json` and set the site `name`, brand colors, navbar, and footer links to match your product.
  </Step>
  <Step title="Replace placeholder copy">
    Swap the content in `index.mdx`, `quickstart.mdx`, and `development.mdx` with your own introductions, tutorials, and FAQs.
  </Step>
  <Step title="Add new pages">
    Create additional `*.mdx` files alongside the existing ones and list them in the `navigation` section of `docs.json`.
  </Step>
</Steps>

## Go live

Commit your changes and push to your default branch. Deploy using your preferred hosting workflow or the Mintlify GitHub integration. Once deployed, your updates will appear automatically.
